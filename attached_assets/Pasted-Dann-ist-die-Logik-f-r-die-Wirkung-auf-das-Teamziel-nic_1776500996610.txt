Doppeldominanz = passend

sondern:

klare Dominanz der Zielkomponente = passend

Zielkomponente nicht tragend = nicht passend

Das passt auch besser zu deiner Aussage:
Wenn zwei Komponenten nah beieinander liegen, ist die Zielwirkung vorhanden, aber eben nicht klar führend.

Saubere Regel
1. Passend

Die Zielkomponente ist klar dominant.

Bedingung:

top1 = Zielkomponente
gap1 > 5
2. Bedingt passend

Die Zielkomponente ist mittragend, aber nicht klar führend.

Das gilt in zwei Fällen:

a) Doppeldominanz
gap1 <= 5
Zielkomponente liegt in Top 1 oder Top 2
b) Balanced / sehr nahe Verteilung
gap1 <= 5
gap2 <= 5

Dann ist keine Zielkomponente klar dominant, aber sie ist auch nicht schwach.

3. Nicht passend

Die Zielkomponente ist nicht klar tragend.

Pseudologik
function computeGoalImpact(personMeta: ProfileMeta, teamGoal: TeamGoal | null) {
  if (!teamGoal) return "nicht bewertet";

  const goalKey = mapGoalToComponent(teamGoal);

  if (personMeta.profileClass === "BALANCED") {
    return "bedingt passend";
  }

  if (personMeta.top1 === goalKey && personMeta.gap1 > 5) {
    return "passend";
  }

  if (
    personMeta.profileClass === "DUAL" &&
    (personMeta.top1 === goalKey || personMeta.top2 === goalKey)
  ) {
    return "bedingt passend";
  }

  return "nicht passend";
}
