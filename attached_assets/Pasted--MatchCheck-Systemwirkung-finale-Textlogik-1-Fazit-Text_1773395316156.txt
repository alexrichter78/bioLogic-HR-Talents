// Nicht geeignet ist nicht automatisch gleich hoher Gap.
// Es ist zu unterscheiden zwischen:
// - strukturellem Fehlfit bei moderatem Abstand => F4 / eher E5
// - deutlicher Gesamtabweichung => F5 / E4 oder E5 je nach Struktur
//
// 3 von 3 darf nur vergeben werden, wenn:
// - fitLabel = Geeignet
// - stimmige Grundausrichtung
// - geringer Anpassungsbedarf
//
// 2 von 3 bei:
// - Bedingt geeignet
// - Entwicklung mit Führung realistisch
//
// 1 von 3 bei:
// - Nicht geeignet
// - oder sehr hoher Entwicklungsaufwand / struktureller Fehlfit
